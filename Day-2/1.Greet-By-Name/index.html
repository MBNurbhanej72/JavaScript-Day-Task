Greet By Name

Exercise 1

  • Create a function greet(name).

  • Inside the function, print a greeting message using the name parameter.

  • Call the function with different argument values (e.g., greet("Alice"), greet("Bob")).

  • console.log("Hello, " + name + "!");

         
        function greet(name) {
          console.log("Hello, " + name + "!");
        }
  
        greet("MohammadAli")
Hello, MohammadAli!
