Simple Timer

Exercise 1

  • Create a simple timer that counts down from a specified number of seconds.

  • Create a counter with start of 10 second in p tag so this counter decrease of each second and If counter value is 0 then stop timer and place of time in p tag show text 'Time up!'.

        
          var counter = 10;
          outputBox.innerHTML = "10";

          const timer = setInterval(() => {
            counter--;
            outputBox.innerHTML = counter;
            if (counter == 0) {
              clearInterval(timer);
              outputBox.innerHTML = "Time up!";
            }
          }, 1000);
