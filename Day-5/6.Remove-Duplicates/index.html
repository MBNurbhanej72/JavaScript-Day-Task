Remove Duplicates (Using includes())

Exercise 1

● Create an array like this const numbers = [1, 2, 2, 3, 4, 1];
● Write a function that takes an array as parameter and print a new array containing only
   unique elements (removing duplicates). Implement a loop and includes() to check for
   existing elements before adding.
● Final result will be console.log(uniqueNumbers); // Output: [1, 2, 3, 4] 


         
        const numbers = [1, 2, 2, 3, 4, 1];

        function removeDuplicateValue(num) {
          const uniqueNumbers = [];

          for (let i = 0; i < num.length; i++) {

            if (!uniqueNumbers.includes(num[i])) {
              uniqueNumbers.push(num[i]);
            }
          }
          console.log(uniqueNumbers);
        }

        removeDuplicateValue(numbers)
[1,2,3,4]
