Exercise 1

  • Create variable like const number = 5;

  • And create for loop of limit 10 and divide number variable to index and each count of loop print this message console.log(`${number} x ${i} = ${result}`);

         
        const number = 5;
    
        for (let i = 1; i <= 10; i++) {
          let result = number * i;
          console.log(`${number} x ${i} = ${result}`);
        }
