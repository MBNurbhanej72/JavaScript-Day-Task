Grade Evaluator

  • Create a variable grade with a letter grade.

  • Assign any one value from this ('A', 'B', 'C', 'D', 'F').

  • Se a switch statement to output a message:
    • "Excellent" for 'A'
    • "Good" for 'B'
    • "Satisfactory" for 'C'
    • "Needs improvement" for 'D'
    • "Failed" for 'F'
    • Include a default case for an invalid grade.
